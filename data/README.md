# video.csvについて
このファイルは国立情報学研究所が公開している「[ニコニコデータセット](https://www.nii.ac.jp/dsc/idr/nico/)」から先頭50万件分を取り出し、学習用に必要最小限の項目に縮小した物です。

研究目的で配布されているデータになります。
詳細は各ページをご覧ください。

* [利用規約](https://www.nii.ac.jp/dsc/idr/nico/documents/dwango-comm-policy.html)
* [利用申請](https://www.nii.ac.jp/dsc/idr/nico/nicocomm-apply.html)
